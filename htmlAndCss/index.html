1. Understanding CSS

CSS complements HTML in that it provides rules for how to display and style the raw (html) content.

CSS can be applied to HTML content inline using the html "style" attribute as part of the opening tag. CSS may also be applied internally within the "header" section of the html document, else as is most common it may be applied externally whereby it is linked to the html document via a 'link' tag in the header.
